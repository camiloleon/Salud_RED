MAPA DE FRAUDES - SOLUCIÓN SIMPLIFICADA CON MARCADORES NEGROS
================================================================

✅ CAMBIOS IMPLEMENTADOS:

1. AGRUPACIÓN POR CIUDAD
   - Ya NO se usan las coordenadas individuales de cada caso
   - Se agrupan todos los fraudes por ciudad
   - Se usan coordenadas fijas del centro de cada ciudad

2. MARCADORES NEGROS ULTRA VISIBLES
   - Color: #000000 (negro puro)
   - Borde blanco grueso: 5px solid #ffffff
   - Sombras múltiples para máximo contraste:
     * 0 0 25px rgba(0, 0, 0, 0.9) - sombra negra principal
     * 0 0 15px rgba(255, 0, 0, 0.6) - resplandor rojo
     * 0 4px 10px rgba(0, 0, 0, 0.5) - sombra de profundidad

3. TAMAÑO DINÁMICO
   - El tamaño del marcador indica el número de casos
   - Fórmula: 20 + (numCasos * 3) píxeles
   - Máximo: 60px
   - Ejemplos:
     * CALI (20 casos) → 80px reducido a 60px (máximo)
     * POPAYÁN (10 casos) → 50px
     * PALMIRA (9 casos) → 47px
     * NEIVA (6 casos) → 38px

4. NÚMERO VISIBLE
   - Cada marcador muestra el NÚMERO de casos en blanco
   - Fuente: 'Roboto Mono', bold
   - Color texto: #ffffff (blanco)

5. FONDO CLARO
   - Tile layer: CartoDB light_all
   - Máximo contraste con marcadores negros

6. POPUP INFORMATIVO
   - Muestra todos los casos de esa ciudad
   - Header negro con nombre de la ciudad
   - Contador grande del total de casos
   - Lista detallada de hasta 8 casos con fecha, canal y asesor
   - Indicador si hay más casos

7. FILTRO POR CANAL
   - Se mantiene la funcionalidad de filtrar por canal
   - Show/hide markers según selección
